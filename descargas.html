2. Descargas

Sólo para equipos con sistema operativo Windows

Este software se requiere para ejecutar la interfase GUI, la cual opera sólo en equipos con sistema operativo Windows, en cualquier versión, ya sea de 32 ó 64 bits, excepto VISTA.

Descargue el software cliente que corresponda con su aplicación. Si no está seguro, verifique con su representante Arex o con el administrador del sistema en su organización.

Se requiere que el sistema operativo tenga aplicadas todas las actualizaciones.

  • Software genérico v.11
  • Software genérico v.10
  • Software genérico Statefree v.11
  • Software personalizado para CMIC/ICIC/FIC/ITC v.11
  • Evaluación de Orientación Vocacional v.11

Debe ejecutar el instalador que se descarga con derechos de Administrador.

Al concluir la instalación quedará un grupo de programas Arex con el ícono correspondiente. Este mismo ícono se crea en el escritorio.
